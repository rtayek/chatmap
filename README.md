# ChatMap

## Run The JavaFX App

From Windows Git Bash:

```bash
./gradlew run
```

The Gradle build configures OpenJFX 25.0.1 and runs `chatmap.ui.ChatMapLauncher`, which starts `ChatMapApp`.
The build uses Kotlin DSL (`build.gradle.kts`) so Eclipse does not need the Groovy editor to open it.

## Run Tests

From Windows Git Bash:

```bash
./gradlew test
```
