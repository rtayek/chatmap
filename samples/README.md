# ChatMap Samples

These files are known-good manual import inputs for the JavaFX UI.

- `plainTextSample.txt`: imports through `PlainTextImporter` as one chat and one `unknown` message.
- `markdownSample.md`: imports through `MarkdownImporter`; the `# ChatMap Markdown Sample` heading becomes the chat title.
- `chatGptSample.json`: imports through `ChatGptJsonImporter` using the supported ChatGPT `mapping` structure.

All samples include the word `ChatMap` so search can be smoke-tested after import.
